# RULES.md - 天气套利交易规则（鱼身版 v2）

## 策略核心（鱼身理论）
- 赚"提前一天的不确定性赔率差 → 第二天赔率差缩小/反转"的利润
- 只对明天/后天标的建仓，禁止当天对当天建仓
- 不追鱼头（太早太不确定），不追鱼尾（临近结算赔率差已消失）
- 目标：长期稳定获利曲线，不追极端收益

## 扫描分工
- 全量扫描（1小时）：找明天/后天新机会 + 检查已有仓位止损/止盈
- 加密扫描（30分钟）：只盯已有仓位止损/止盈，不开新仓
- 城市：上海（ZSPD）、达拉斯（KDFW）、伦敦（EGLL）、首尔（RKSI）、惠灵顿（NZWN）、米兰（LIMC）
- 数据源：TWC API（WU 同源）+ Polymarket Gamma/CLOB + NOAA METAR
- sigma 按时间分层：当天 0.8，明天 1.0，后天及更远 1.5

## 建仓规则
### 1）先定中心档
- 先用 TWC/WU 预报确定明天或后天的预测最高温中心档
- 不机械只做中心档，还要同时看中心档的相邻两档（中心档 +1 / -1）

### 2）判断市场会提前交易哪一边
#### 偏向 +1 档
如果同时满足：
- 城市处于升温季节
- 明天升温速度快
- 中午前后就摸到中心档
- 天气支持继续升温：晴 / 少云 / 无雨 / 无大风

则判断：
- 市场更容易提前抬高“中心档 +1”价格
- 如果 +1 档当下价格还不高，优先考虑提前一天做这档

#### 偏向 -1 档
如果同时满足：
- 城市处于降温季节
- 明天升温速度慢
- 峰值出现较晚
- 天气压制升温：有雨 / 云多 / 风大

则判断：
- 市场更容易提前抬高“中心档 -1”价格，或压低中心档本身
- 如果 -1 档价格还没充分反映，优先考虑这档

### 3）只做“预期未充分反映”的那一档
必须同时满足：
1. 只做明天/后天标的，禁止当天标的
2. edge ≥ 20%
3. 盘口价 10%-90%
4. 订单簿深度足够（实际成交后仍满足最低 edge）
5. 赔率位置过滤：
   - BUY_YES：YES 在 10%-35%（优先），35%-40%（谨慎），其他不做
   - BUY_NO：NO 在 10%-35%（优先），35%-40%（谨慎），其他不做
6. 同标的止损后当天禁止重入（除非出现新的硬信息变化：TWC 预报跳变 / METAR 改变判断）
7. 单笔上限 $20，单城市总暴露不超过总资金 60%
8. 同城市 + 同日期最多 2 笔
9. 今天标的一律禁止新建仓

### 4）建仓方向
- 当判断市场后面会继续抬高某一档、但当前价格还偏低时，做 BUY_YES
- 当判断市场现在给某一档价格过高、后面会回落时，才做 BUY_NO
- 当前阶段：优先做 BUY_YES，谨慎做 BUY_NO，直到新 BUY_NO 样本重新积累够

## 补仓规则
- 仅允许鱼身仓（明天/后天标的）
- 最多补 1 次
- 补仓后总成本不超过单标的上限 $20
- 补仓 ≤ 原仓位 33%
- 距离结算 ≥ 24 小时
- 模型没有恶化超过 5%，edge 仍 ≥ 20%
- 禁止：当天标的、接近结算窗口、盘口已快速收敛、止损信号已出现
- 原则：补仓只给真正的鱼身仓，不给快兑现的仓

## 止盈规则
满足任一即止盈/减仓：
1. edge 衰减止盈：当前 edge ≤ 入场 edge 的 40%，且仓位仍盈利
2. 盘口明显向模型收敛，原本交易的预期已经被价格反映
3. 结算日进入当地升温窗口（6:00–14:00）且浮盈：先卖一半，剩余继续观察
4. 单笔浮盈 ≥ 30%，且盘口已明显兑现预期

原则：
- 我们交易的是“预期增强 → 价格反映”这段，不是等预期最终兑现
- 价格已经反映预期时走，不把鱼身拖进鱼尾回吐

## 止损规则（任一触发即执行）
1. 模型翻转止损：currentEdge < -10%
2. 连续两轮 negative edge，或单轮 edge < -8%
3. 浮亏超过 50%
4. 盘口极端化（BUY_NO 但 Yes>90%，BUY_YES 但 Yes<10%）
5. METAR 实测已确认对赌阈值
6. WU/TWC 预报偏移 ≥ 2°C
7. 原本交易的路径预期失效：峰值出现时间明显后移、升温速度明显转弱、或天气条件从支持变成压制
8. 相邻档替代止损：若精确温度 BUY_YES 仓位当前 edge 已掉到 10%以下，且相邻 ±1 档连续两轮明显更强（相邻档 edge 至少高 10%），则提前退出

原则：
- 预期不成立就走
- 错了就认，不和市场硬拗
- 绝不等预期兑现的最后关口

## 仓位控制
- 单标的总仓位上限：$20
- 单次补仓 ≤ 原始仓位 50%
- 单城市总暴露建议不超过总资金 60%

## 复盘标准
每笔交易看 4 件事：
1. 是不是鱼身单（建仓日 < 结算日）
2. edge 是否真实
3. 赔率位置是否舒服
4. 是否按规则退出
非鱼身单单独归类，不能拿来证明鱼身策略有效

## 汇报纪律（铁律）
- 所有交易记录 100% 来自 portfolio.json，不得编造/修饰
- 盈亏如实反映，亏了就说亏了
- 不确定的标注"不确定"，不编造数据
- 绝对客观、理性，零杜撰、零吹水
