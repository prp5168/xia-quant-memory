- 平均每笔止盈赚 $0.76 → 持有到结算平均赚 $6.84
- 60% edge captured 止盈是最大的"少赚"来源

#### 盈亏比问题
- 当前盈亏比: 0.111（赢均 $0.61, 输均 $-5.49）
- 要盈亏平衡需要 90% 胜率
- 实际胜率 73.5%，远不够
- **根本原因：止盈太早（赚太少）+ 止损太敏感（不该止的止了）**

### 结论
1. **tail 桶的 V1 止损规则不适合 V2**：尾部桶本身就是"我不知道具体几度，但一定不是这个极端值"的逻辑，预报偏移 2°C 对尾部桶影响极小
2. **止盈规则过于保守**：60% edge captured 止盈把 $6-11 的结算利润压缩到 $0.3-1.0
3. **如果 V2 tail 桶直接持有到结算**，19 笔样本的 PnL 从 -$22.66 变成 +$98.54
4. 这验证了 V2 策略（BUY_NO tail+near only）的选股方向是对的，问题出在退出规则
